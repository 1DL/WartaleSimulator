# WartaleSimulator
Wartale Simulator is a gamified simulator of Wartale server of Priston Tale.
Instead of raw numbers and text, this simulator allows you build your character and check all his status in a fun, visual appealing way.
Then, you can simulate battle and check what is your accuraccy chance, damage output and defensive ability.

You can check all itens stats that are currently released in the real game, and set them on your character.
While doing so, you can simulate how much gold you will spend, how much sheltons required, and a lot more.

Wartale Simulator primary goal is to make the process of Game Balance to be easier, with instant visual feedback.

Further releases will have a feature to customize a gear and skill, to help players discussing and find a good balance for each class.

Hope you enjoy!
